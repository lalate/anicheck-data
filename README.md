# 🛡️ アニちぇっく・データ要塞 (anicheck-data)

このリポジトリは、アニメ番組表アプリ「アニちぇっく」で使用される放送スケジュールデータを管理しています。
すべてのデータはJSON形式で公開されており、誰でも自由に修正・提案（Pull Request）を送ることができます。

## 🏰 データ構造

- `current/`: 現在放送中のアニメデータ（アプリが直接参照する聖域）
  - `watch_list.json`: 監視対象の全アニメリスト
  - `daily_schedule.json`: 本日の統合放送スケジュール
  - `{anime_id}_master.json`: 作品の基本情報（キャスト、スタッフ、アフィリエイトリンク）
  - `{anime_id}_episode.json`: 各話の詳細情報（サブタイトル、あらすじ、関連グッズ）
  - `{anime_id}_broadcast.json`: 具体的な放送時間
- `archive/`: 過去に放送されたシーズンの記録
- `scripts/`: データ収集・検証用Pythonスクリプト

## 🛠️ 職人の皆様へ（データの修正方法）

地方局の細かな時間変更や、放送休止情報を発見した場合は、以下の手順で修正をお願いします。

1.  修正したい作品のJSONファイルを `current/` フォルダ内から探します。
2.  GitHubの編集ボタン（ペンアイコン）を押し、内容を修正します。
3.  変更内容を記載して **Pull Request** を送信してください。
4.  GitHub Actionsが自動でJSONの構文チェックを行い、問題なければ運営がマージします。

## 🤖 自動更新システム

このリポジトリは、毎日 午前8時 (JST) に Grok AI を使って自動的に最新の放送スケジュールとあらすじを収集しています。
AIの抽出ミスを発見した場合は、ぜひ手動での修正をお願いします。
